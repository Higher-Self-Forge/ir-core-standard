# Changelog

All notable changes to **IR v1.0 Core Standard** will be documented in this file.

## [1.0.0] — 2026-01-18

### Added
- Initial normative schema set:
  - Graph Node, Constitution Manifest, Audit Entry, Consent Record (FPIC), Boundary Card
- Conformance checklist (IR-Conformant, IR-Conformant+)
- Governance and consent templates (Council Charter, Decision Register, Custodian Declaration, FPIC record, Consent materials checklist)
- Implementation and auditor guides
- Standard summary document
