# Lista de verificación de materiales de consentimiento (Apoyo FPIC)

> Plantilla localizada (es-PE). Este documento es una traducción informativa del estándar IR. Una vez adoptado por una comunidad, es normativo para la gobernanza de ese despliegue, pero no sustituye la especificación normativa en inglés.

<!--

## 1. Claridad de propósito y alcance
- [ ] La declaración de propósito es clara y no técnica
- [ ] Las categorías de datos están listadas con ejemplos
- [ ] Las actividades de tratamiento están listadas (recolectar/almacenar/entrenar/inferir/compartir)
- [ ] Beneficiarios y límites de acceso están claros
- [ ] Se explica el plan de retención y eliminación/salida

## 2. Riesgos y salvaguardas
- [ ] Se explican los riesgos conocidos (mal uso, reidentificación, sesgos, seguridad)
- [ ] Se explican las salvaguardas (límites, registro de auditoría, controles de acceso)
- [ ] Se explican las limitaciones (precisión, incertidumbre, modos de falla)
- [ ] Se entrega un canal de quejas / incidentes

## 3. Derechos y opciones
- [ ] Se explica el mecanismo de revocación
- [ ] Se admiten condiciones y restricciones
- [ ] Se explican los disparadores de consentimiento (qué cambios requieren re-consentimiento)
- [ ] Se discuten alternativas (incluyendo rechazar la participación)

## 4. Accesibilidad y lengua
- [ ] Existen materiales en los idiomas pertinentes (p. ej., es-PE, quz)
- [ ] Hay apoyo de interpretación si se requiere
- [ ] Hay apoyos visuales accesibles (opciones para baja alfabetización)

## 5. Evidencia y registro
- [ ] Se registran actas de reunión (o equivalente legible por la comunidad)
- [ ] Se capturan firmas/huellas (según corresponda)
- [ ] Las referencias de evidencia incluyen hashes cuando sea posible
- [ ] Se crea un Registro de Consentimiento legible por máquina (REQUERIDO para IR-Conformant; recomendado incluso para pilotos)
